## Design Principles (Reminder)

As we build new features, remember:

1. **Glanceable** - Info visible from across the room
2. **Collaborative** - Everyone feels ownership
3. **Motivating** - Completing tasks feels rewarding
4. **Fun** - This is a family app, not corporate software
5. **Simple** - Features should be intuitive, not overwhelming
